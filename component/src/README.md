# component/src/

Исходники Web Component `<createui-icon>`.

## Файлы

```
src/
└── icon.ts       # Класс кастомного элемента, регистрация, экспорты, типы
```

Все публичные экспорты пакета собираются из `icon.ts`:

- Регистрация кастомного элемента `<createui-icon>` в `customElements`.
- Функция `createIcon()` — программное создание элемента.
- Тип `TIconName` — объединение имён всех иконок текущей версии Lucide (генерируется при сборке).
- Интерфейс `IIconProps` — форма атрибутов компонента.
