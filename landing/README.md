# landing/

Лендинг компонента на `icons.createui.dev/`. Astro / статика.

## Файлы

```
landing/
├── src/           # Исходники (страницы, компоненты, стили)
└── dist/          # Собранная статика (деплоится в /var/www/icons-landing/)
```

`dist/` **не коммитится** — собирается на CI командой `npm run build` перед деплоем.

## Содержимое

- **Описание компонента** — что такое `<createui-icon>`, установка, использование.
- **Примеры кода** — HTML, SolidJS, React, Vue.
- **Каталог иконок** — поиск по имени, превью всех иконок текущей версии.
- **Changelog** — список версий с датами, количество новых/изменённых/удалённых иконок.
- **Текущая версия** — отображается на странице, обновляется CI при синхронизации.

## Раздача

Nginx отдаёт `dist/` как статику по `location /`. Без обращения к Go-серверу.
